PPG-based Emotion Detector

Technologies: Python, C/C++, neurokit2, ESP32, MAX30102

  • Developed a PPG-based emotion detection system using Siamese Neural Networks with TensorFlow/Keras, leveraging physiological signals to classify emotional states.
  • Applied signal processing and similarity metrics such as Euclidean distance, Pearson correlation, and Dynamic Time Warping (DTW) with libraries like NumPy, SciPy, FastDTW, and Pandas.
  • Integrated hardware components including ESP32, MAX30102, and MAX32664 using Arduino IDE and C/C++ for real-time data acquisition and processing.
